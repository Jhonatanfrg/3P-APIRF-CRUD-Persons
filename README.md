# AngularPersonCRUD

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 12.2.4.

## Primero debe poner en marcha el siguiente proyecto, el cual corresponde al servidor:

https://drive.google.com/drive/folders/1YRLwCkLxvUY4WcPp6Y4fW3tidikHkthS?usp=sharing , descargar y correr `npm install`, luego `node server.js` para correr el servidor y dejarlo en el puerto 8080.

## Development server

Corra `ng serve --port 8081` para iniciar el proyecto. Ingrese a `http://localhost:8081/` para observar el proyecto.

## Build

Run `ng build` para hacer el build del proyecto. 

## Running unit tests

Run `ng test` para correr los test unitarios usando karma [Karma](https://karma-runner.github.io).
